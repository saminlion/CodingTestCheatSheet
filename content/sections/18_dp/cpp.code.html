// 피보나치 수열에서 n번째 값을 구하는 DP 알고리즘 (Bottom-Up 방식) // 시간복잡도: O(n) vector dp(n + 1); // dp[i]: i번째 피보나치 수 저장 dp[0] = 0; // F(0) = 0 dp[1] = 1; // F(1) = 1 // 점화식 구간 (핵심 로직) // ▶ dp[i] = dp[i-1] + dp[i-2] // ▶ 이전 두 항으로부터 현재 값을 유도 for (int i = 2; i <= n; i++) dp[i] = dp[i - 1] + dp[i - 2]; // ← 점화식 적용 // dp[n]이 최종 결과 F(n)
