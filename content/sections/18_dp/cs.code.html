// 피보나치 수열(Fibonacci)에서 n번째 항을 구하는 DP(동적 계획법) 방식 // 시간복잡도: O(n) int[] dp = new int[n + 1]; // dp[i]: i번째 피보나치 수 저장 배열 dp[0] = 0; // 초기 조건 1: F(0) = 0 dp[1] = 1; // 초기 조건 2: F(1) = 1 // 점화식 적용 구간 (핵심 로직) // ▶ dp[i] = dp[i - 1] + dp[i - 2] // ▶ 이전 두 항의 값을 더해 현재 항을 유도 for (int i = 2; i <= n; i++) { dp[i] = dp[i - 1] + dp[i - 2]; // ← 점화식 적용 (현재 값 ← 이전 두 값으로 계산) } // 결과적으로 dp[n]이 F(n)에 해당
